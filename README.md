# Snake

Ein einfaches Snake-Spiel fuer den Browser.

## Spielen

Oeffne `index.html` im Browser und steuere die Schlange mit den Pfeiltasten oder WASD.

Auf kleinen Bildschirmen erscheinen zusaetzlich Touch-Buttons.

## Dateien

- `index.html` enthaelt die Oberflaeche.
- `style.css` gestaltet das Spiel.
- `script.js` enthaelt die Spiellogik.
